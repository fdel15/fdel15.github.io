One of the major differences between coding in Ruby and coding in Javascript is the way each language handles loops.

Loops are a common part of every programming language. They allow a program to repeat the same action a set number of times or until a specified condition is met. Loops are an essential part of keeping your code DRY.

There are many different ways to create loops in Ruby. You can use specified keywords or methods. One way to perform an action a set number of times is to use the .times method, for example if you want print out a sentence three times:

3.times {p “I love Ruby!”}

If you want to create a loop until a specific condition is met you can youse the while keyword. The following loop will continue until the variable i is greater than or equal to 3.:

  • i = 0
  • while i < 3
  • p “I love Ruby!”
  • i += 1
  • end

Ruby also has a bunch of different methods to loop through a data structure. Some of the methods include .each, .map, and .inject. These methods make it simple to find the sum of an array of integers:

array = [1,3,5,6]

array.inject{ |sum, num| sum + num} => 15

JavaScript on the other hand only offers two keywords for creating loops. They are the keywords for and while. These two keywords are powerful enough for you to create all of the different kinds of loops that are available in the Ruby language, you just have to work a little bit harder as a programmer.

The while loop in JavaScript behaves much like the while loop in Ruby, however the syntax is a little more detailed:

  • var n = 1;

  • while (n <= 3) {
  • console.log(“I love JavaScript!”);
  • n += 1
  • }

For loops are useful for looping a given number of iterations, much like the .times method in Ruby. For loops take in 3 parameters: the first parameter defines a variable, the second parameter defines the number of iterations, and the third parameter increases or decreases the variable after each iteration.

  • for (var i = 0; i < 3; i++) {
  • console.log(“I love JavaScript”)
  • }

If you want to loop through an Array in JavaScript you can use a modified version of the for loop called the for (x in y) syntax where x is a variable representing an increment able value and y represents an object with a length property such as an array. Here’s our previous Ruby example done in JavaScript:

  • sum = 0
  • array = [1,3,5,6]
  • for (i in array) { sum += array[i]} => 15

That is a brief comparison of the differences of loops between Ruby and JavaScript. Hopefully you have a little bit better understanding after reading this post.
