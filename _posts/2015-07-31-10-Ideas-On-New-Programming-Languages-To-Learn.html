It is impossible to learn everything there is to know about programming and technology. The subject is too broad and complex. The fun part about this is that programmers are always learning new technologies and languages. Being a life long learner is part of the job. Here are some ideas on what I should focus on learning next:

The Ideas

  1. PHP
    • The CodeIgniter in particular because of the Buffer job description
  2. ReactJS
    • From what I currently hear/know, this is my best guess at the next dominant JS framework
  3. Objective-C
    • Build native iOS apps
  4. Python
    • Should work well with my Math degree.
  5. R
    • Big data analysis
  6. Go
    • Great community and seems to be a popular transition for Ruby developers.
  7. Java
    • Java code (jobs) is everywhere and will allow me to build Android apps
  8. Arduino/Raspberry pi
    • Understand the connection between hardware and software
  9. JavaScript
    • I can still get much better at understanding JS and design patterns
  10. Backbone.js
    • Framework is listed on Buffer job description
