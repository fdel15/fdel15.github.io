When learning CSS I found myself asking what the heck is a class, an ID, and why should I care? In this blog post I am going to expalin the difference between ID's, Classes, and when you should use each.

A class is a group of elements that are the same or similar. The most important thing to remember about classes is that they are not unique. This means that you can use the same class on multiple elements and you can have as many elements as you want in a class.

An ID is a singular identifier of one HTML tag. The most important thing to remember about ID's is that they are unique. Each element can have only one ID and each page can only have one element with that ID. Your code will not pass validation if you use the same ID on more than one element. One of the main benefits of using ID's is that they can be accessed directly through a hash value.

So when should you use each one? Any time you want to apply the same styling to multiple elements you want to use a class.
